About Us

This is the About page of our test website. We're using this site to demonstrate and test the chat widget navigation warning functionality.

When you have an active call and try to navigate between pages on this non-SPA website, you should see a warning modal that asks if you want to open the new page in a new tab to keep your call active.

This feature is particularly useful for websites that don't use client-side routing (SPAs), as navigating away from the page would normally end the call due to the API connection being terminated.

The navigation warning system automatically detects whether a site is an SPA or not, and only shows warnings on non-SPA sites where navigation would actually end the call.
